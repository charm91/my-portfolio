# ngu-wah-portfolio

Personal portfolio site for Ngu Wah Aung (Charm), Senior Product Designer.

## Table of Contents

- [Install](#install)
- [Usage](#usage)
- [Contributing](#contributing)

## Install

Node.js 18+ required.

```bash
npm install
```

## Usage

**Development**

```bash
npm run dev
```

Runs on [http://localhost:3002](http://localhost:3002).

**Production build**

```bash
npm run build
npm run start
```

**Lint**

```bash
npm run lint
```

**Tests**

```bash
npm test              # run all tests once
npm run test:watch    # watch mode
```

**Environment**

No `.env` file required for local development. The site URL defaults to `https://www.nguwahaung.com/` if `NEXT_PUBLIC_SITE_URL` is not set.
